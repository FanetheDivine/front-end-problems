# 问题
自带的纯色blue,green等显示效果较差 尤其是暗色背景  
# 解决
使用更好的色彩  
在灰色背景下演示  
## 红色
``` css
.pure-color{
    color:red;
}
.soft-color{
    color:rgb(219, 68, 68);
}
```
效果  
![image](images/红色.png)
## 绿色
``` css
.pure-color{
    color:green;
}
.soft-color{
    color:rgb(82, 196, 26);
}
.dark-soft-color{
    color:#08656e
}
```
效果  
![image](images/绿色.png)
## 蓝色
```css
.pure-color{
    color:blue;
}
.soft-color{
    color:rgb(113, 210, 250);
}
.light-soft-color{
    color:rgb(71, 228, 238);
}
```
效果  
![image](images/蓝色.png)
